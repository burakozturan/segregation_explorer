Segregation metric: Theil H

Theil H measures how much knowing which source someone visited reduces uncertainty about their demographic group. Formally:

H / Hmax  =  Σu   (nu / N)  ×  Σg   (nug / nu)  ×  ln [ (nug / nu) / πg ]   /   −Σg   πg  ln πg

where n𝑢 is traffic at source u, n𝑢ɡ⃗ is traffic from group g at source u, and πɡ⃗ is the population share of group g. H = 0 means all sources have identical audience composition. H = 1 means each source is visited exclusively by one group.

Weighting artifact (top panel)

Volume weighting counts every click equally, so heavy users dominate. Person weighting normalizes each user to contribute equally regardless of activity level. The gap between them is:

gap  =  Hvol − Hper  ≈  N  ×  Cov( activity sharei , sortednessi )

The gap is nonzero only when activity is concentrated AND heavy users are more sorted than typical users. Concentration alone produces no gap if heavy and light users are equally sorted.

The simulation draws an activity level for each of the 2,000 users from the chosen distribution. Users are then ranked by activity: the top X% (the super-sharer fraction) receive homophily HS and the rest receive HG. For each visit, a random draw determines whether the user visits a same-group domain (with probability H) or any domain chosen uniformly at random (with probability 1−H). Super-sharers are assigned by activity rank, so both Democrats and Republicans can be super-sharers.

Simulation note. All three metrics (Theil H, GS S, Dissimilarity D) are computed from a full simulation: 2,000 users, S/N = 0.1 (safe zone, negligible fragmentation), mean 18 visits per user. Volume weighting counts each visit as 1; person weighting gives each visit weight 1/vi so every user contributes equally. Domains are split equally between groups (binary partisan structure), so absolute levels are an upper bound relative to real mixed-audience domains. The gap is a simulated result.

Fragmentation artifact (bottom panel)

When sources outnumber users (S/N > 1), most sources receive a single visitor and appear perfectly segregated by construction, regardless of true partisan sorting. The metric inflates toward 1 as S/N rises. The simulation holds homophily fixed and varies the number of sources, showing how the person-weighted Theil departs from the true underlying level as S/N grows.

Simulated with 3,000 users. The qualitative shape (inflation above S/N = 1) is robust; absolute levels depend on panel size and activity distribution.

Distribution calibration

The concentration slider controls the target share of total activity owned by the top X% of users (where X is the super-sharer percentage). Both power-law and log-normal distributions are analytically calibrated to produce the same top-X% share at each slider position, so switching distributions does not change the headline concentration statistic.

Power-law uses a Pareto distribution; log-normal uses a two-parameter log-normal. Equal activity assigns identical browsing volume to all users.
